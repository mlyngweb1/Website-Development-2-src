jQuery Part One

2018 Teaching Team: Rosanne Birney and Mary Lyng

    Lecture

    In this lecture, you will learn about jQuery, a JavaScript library that simplifies interactions with HTML pages.

    jQuery Selectors

    This lab will introduce you to the jQuery JavaScript library.

    jQuery Filters

    This lab introduces filters, which allow you to filter your selections based on certain characteristics.

    Adding content

    jQuery provides many functions for manipulating elements and content on a page, from simply replacing HTML, to precisely positioning new HTML in relation to a selected element, to completely removing tags and content from the page.

    Modifying CSS properties

    jQuery can also be used to change the CSS properties of elements.

    Tag attributes

    jQuery can also be used to get, set or remove HTML attributes.

    Exercises

    In these exercises we will use jQuery to manipulate HTML elements on a web page.
