
# charAt()

Syntax: **charAt(index)**

The **charAt()** method returns the **character** which is at **position index** in the string. 

~~~
<script>
 document.write("<h1>charAt()</h1>");
 var name= prompt("Enter your name", "");
 document.write("The first character is "+ name.charAt(0));
</script>
~~~
